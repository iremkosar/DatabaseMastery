🚀 ASP.NET Core & MongoDB ile Lojistik Yönetim Sistemi

Bu proje, ASP.NET Core MVC ve MongoDB kullanılarak geliştirilmiş, gerçek senaryoya dayalı bir Kargo Takip ve Lojistik Yönetim Sistemi uygulamasıdır.

🔧 Kullanılan Teknolojiler

ASP.NET Core (MVC)

MongoDB (NoSQL – Document Database)

AutoMapper

Clean Code yaklaşımı

Katmanlı mimari yapı

🧩 Proje Yapısı

Proje içerisinde:

Controllers

DTOs

Entities

Services

Mapping (AutoMapper)

Settings

ViewComponents

katmanlı ve sürdürülebilir bir mimari ile yapılandırılmıştır.

📦 Özellikler

Dinamik kargo oluşturma ve yönetimi

Tracking geçmişi (embedded document yapısı)

Anlık durum takibi

Admin panel üzerinden içerik yönetimi

Maliyet hesaplama modülü

Bu projede amacım yalnızca CRUD işlemleri yapmak değil; gerçek hayata uygun, ölçeklenebilir ve temiz kod prensiplerine uygun bir sistem tasarlamaktı.
